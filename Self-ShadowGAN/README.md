## Self-ShadowGAN: Learning to Remove Shadows from a Single Image
This is the implementation of the proposed Self-ShadowGAN.


## Requirements
- python 3.7+
- pytorch 1.4+ & tochvision
- math
- scipy
- scikit-image
- matplotlib
- pillow
